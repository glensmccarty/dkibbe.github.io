Customizing the Bash Shell

Assignment Objectives

Preparation

This assignment is for CentOS 7.

Create a New User

Create a new user.

# useraadd tux
# passwd tux

Login as user tux.

# su -l tux

Create Files

Create an directory, an ordinary file and a symbolic link.

$ mkdir MyStuff ; touch foobar ; ln -s foobar foo

Custom Prompt

Follow the instructions in the link under Resources to create a custom prompt.

Custom Colors

Copy /etc/DIR_COLORS to ~/.dir_color.

Follow the instructions in the file to change the colors on the directories sho they show up better.
