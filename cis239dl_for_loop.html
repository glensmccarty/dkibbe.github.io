For Loops

Introduction

Often you want to repeat an action on a list of files. A for loop will do this. For this assignment you will create a for loop that will use the tr command to convert upper case file names to lower case.

Term You Should Know

  • tr
  • for
  • seq
  • touch
  • Character classes

Preparation

  1. Create a directory in your home directory named ~/temp and change to that directory.
  2. Use the touch command to create the files.
$ touch FILE{1,2,3,4,5,6,7,8,9}.txt

seq Command

You can also create the files using a for loop and the seq command.

$ for i in $(seq 1 99); do touch FILE"$i".txt; done

This will create a sequence from 1 to 99. Each instance of the variable i is replaced by the output of the seq command.

The tr Command

The tr command can translate a string of characters. Here you will use tr to convert file names from upper to lower case. try the command by its self before you use it in a script.

$ touch FILE{1,2,3,4,5}
$ ls FILE?
FILE1
FILE2
FILE3
FILE4
FILE5

The syntax for tr is command, options, set1, set2

$ ls FILE* | tr [:upper:] [:lower:]
file1
file2
file3
file4
file5

for Loop

The syntax for the for loop looks like this:

for i in <list>
do
<do something to $i>
done

There is nothing special about the variable $i. You can call it what ever you want just be careful not to use a variable name already in use.

for myvar in file?
do echo "$myvar"
done

Get in the habit now of quoting variables so that they will always expand correctly.

What to Submit

Write a script that uses a for loop to remove the files you created. For 5 extra points use a for to remove only the odd numbered files leaving the even numbered files untouched.
