Add a Disk

Using fdisk

Open a terminal
Use fdisk to find the new disk
$ sudo fdisk -l
/dev/sdb doesn’t have a partition table yet
m for menu
p print partition table
n for new partition
use the defaults
p again to see partition table with default Linux type (83)
t for new type
l for list
c for W95 FAT32 (LBA)
p to show change
w to write changes and sychronise the disks

Format the disk

The disk isn’t useable until formated

fdisk -l
note disk is /dev/sdb1 importtant!
sudo mkfs Tab Tab to see file tyoes

sudo mkfs.vfat to format as FAT
sudo mkfs.vfat /dev/sdb1

As soon as formatted mounted under /media/tux/
mount command shows it

Create mount point

umount
mkdir
mount -o rw /dev/sdb1 /mnt/data
