Loops

Bash supports three types of loops.

  • For
  • While
  • Until

For Loops

Syntax

for arg in [list]
do
action
done

Example: Lists

for fruit in apple orange pear
do
echo $fruit
done

Example: On One Line

$ for fruit in apple orange pear; do echo $fruit; done

Example: Command Expansion

for file in *
do
echo $file
done

While Loops

Syntax

while [ condition ]
statements
done

Example

#!/bin/bash  
myvar=0

while [ $myvar -ne 10 ]
do
echo $myvar
myvar=$(( $myvar + 1 ))
done

Example 2

#!/bin/bash

NUM=0
MAX=20

while [ "$NUM" -gt "$MAX" ]
do 
echo $NUM
let "NUM += 1"

done

Until Loops

Syntax

until [ condition ]
statements
done

Example

#!/bin/bash

myvar=0

until [ $myvar -eq 10 ]
do
echo $myvar
myvar=$(( $myvar + 1 ))
done

Case Statements

Syntax

case expression in
pattern1 )
statements ;;
pattern2 )
statements ;;
...
esac

Example 1

#!bin/bash

# Set variables

numb=0
numbok=0

until [ $numbok -eq 1 ]; do
echo -n "Pick a number between 1 and 10: "
read numb 
numbok=1

case $numb in 
1) echo "One is the loneliest number that you'll ever do." ;;
2) echo "Two can be as bad as one... It's the loneliest number since the number one.";;
3) echo "Three can be as bad as two... It's the loneliest number since the number two." ;;
4) echo "Four has enough friends to never be lonely." ;;
5) echo "Five has enough friends to have enemies." ;;
6) echo "Six is afraid of Seven because seven ate nine." ;;
7) echo "Seven is hungry for more friends." ;;
8) echo "Eight is enough" ;;
9) echo "Nine is afraid of being eaten by seven." ;;
10) echo "Ten is very popular." ;;
*) echo "Nice try.. how about a number between 1 and 10?"
numbok=0 ;;
esac
done
