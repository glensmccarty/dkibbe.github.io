* Assignment Objectives

  IRC is one of the oldest real-time ways to communicate on the Internet.  Most Linux distributions and Free and Open Source software projects have a channel on Freenode.  In this assignment you will log into a channel and post an entry.

/When you have successfully completed this assignment you will be able to:/

- Use a web browser to log into Freenode.
-  Join an IRC chat channel.
- Post a chat.
- Use common IRC commands.

* Terms You Should Know

- IRC :: Internet Relay Chat
- Channel :: A channel is a chat room and begins with the *#* character.  Example: *#ubuntu-us-az*.
- IRC commands :: IRC command begin with the */* character.  Example: */join*
- Freenode :: Freenode hosts thousands of channels related to free Open Source software.

* Preparation

  This assignment can be done using any web browser.  Go to [[http://irchelp.org][IRC Help]] and click on /Get Started/.  Read /The IRC Prelude/.

* Using IRC

  1. Go to the Freenode site.
  2. Click on /webchat/ in the left hand column.
  3. Login using a nickname of your choosing.
  4. Enter /cis126dl/ as the channel name.
  5. Post an entry which includes your real name.
  6. Log out of the Channel.

  If the nickname you choose is registered you will get that name followed by an underscore. /Example:/ tux_ instead of tux.

* What to Submit

  This assignment is done during the class period.  I will verify that you logged into the channel.
