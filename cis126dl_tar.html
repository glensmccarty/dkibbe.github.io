Archiving & Compressing Data

Introduction

The tar command was originally designed for backing up data to tape. While it is still used for that it is more commonly used to create archives of file called “tarballs.”

Learning Objectives

When you have successfully completed this assignment you will be able to:

  1. Create a Tar archive with and without compression.
  2. Extract files from a Tar archive
  3. List files in a Tar archive
  4. Use Gzip and Bzip2 to compress files.
  5. Identify the advantages of Xz compression.

Tape Archive (TAR)

Basic Syntax

tar option(s) archive_name file_name(s)

Common Options

Option Description
c Create archive
x Extract archive
t List files in archive
u Append newer files
r Add files to archive

Options can be preceded by a single dash (-) but it is more common to omit the dash.

Create an Archive

Create a new archive including all files named foo.

$ tar cf foo.tar foo*

List the Contents of an Archive

The t option will list the contents of an archive.

$ tar tf foo.tar
