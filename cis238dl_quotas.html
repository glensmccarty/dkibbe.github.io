Preparation

Create a new CentOS 7 virtual machine with a 12 GiB virtual hard disk. /home should be on a separate 1 GiB Logical Volume. You can take the space needed from swap. Set the filesystem type to ext4. Make sure networking is active on boot.

During install create a user account.

Enable Quotas

yum install quota

Edit the etcfstab` file.

dev/sda1       /home               ext4    errors=remount-ro,usrjquota=aquota.user,grpjquota=aquota.group,jqfmt=vfsv0 0       1

Remount /home

# umount /home
# mount /home
