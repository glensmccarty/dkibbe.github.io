git

Introduction

Git is a very popular version control system created by Linus Torvalds. It is the system used by the Linux kernel developers and powers the popular social coding site, GitHub.

Installation

# yum install git

Configuration

Git requires a minimum of configuration.

$ git config --global user.name "dennisk"
$ git config --global user.email "dennisk@whatever.com"

Initializing a Git Repo

$ git init Project1
Initialized empty Git repository
$ cd Project1

.git Directory

Git specific files are keep here.

$ tree .git

Add a File to git

Before you can manage a file git must know about it.

$ touch README.md
$ git status
nothing added to commit but untracked files present (use "git add" to track)

$ git add README.md

Commits

After modifying a file the changes can be committed.

$ echo "Add text." > README.md

$ git commit -m"First commit"
[master (root-commit) a98ad5f] First commit

1 file changed, 0 insertions(+), 0 deletions(-)
create mode 100644 - - README.md

Change a Commit

$ git commit --amend -m"Added some text to the file."

Undo Last Commit

$ git cehckout -- README.md

Revert an earlier Commit

$ git log --oneline

b62a6c9 Revert "Removed Section 2."
f820500 Removed Section 2.
09775dc Added Section 1 to README.
6ac8f2b Added README.

$git revert f820500

$ git log

Author: dennisk <dennisk@whatever.com>
Date:   Thu Mar 3 16:55:10 2016 -0700

Git ignore

git can ignore files you don't wish to track.

$ cat .gitignore
*.[oa]
*~

The rules for the patterns you can put in the .gitignore file are as follows:

  • Blank lines or lines starting with # are ignored.
  • Standard glob patterns work.
  • You can start patterns with a forward slash (/) to avoid recursivity.
    You can negate a pattern by starting it with an exclamation point

Branches

Git creates a default master branch.

You can create a new branch with this command.

$ git branch testing

Then switch to that branch.

$ git checkout testing

or do both at once.

$ git checkout -b new_branch

Delete a Branch

When you no longer need a branch you can delete it.

$ git branch -d new_branch

Merge Branches

Change to the branch you want to merge into then:

$ git checkout master
$ git merge new_branch

See Last Commits

$ git branch -v
  docs   11ed404 Added Section 3.
* master b2b0f7f Merge branch 'docs'
