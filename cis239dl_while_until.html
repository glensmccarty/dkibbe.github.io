Loops

Bash supports three types of loops.

  • For
  • While
  • Until

For Loops

Syntax

for arg in [list]
do
action
done

Example: Lists

for fruit in apple orange pear
do
echo $fruit
done

Example: On One Line

$ for fruit in apple orange pear; do echo $fruit; done

Example: Command Expansion

for file in *
do
echo $file
done

While Loops

Syntax

while [ condition ]
statements
done

Example

#!/bin/bash  
myvar=0

while [ $myvar -ne 10 ]
do
echo $myvar
myvar=$(( $myvar + 1 ))
done

Example 2

#!/bin/bash

NUM=0
MAX=20

while [ "$NUM" -gt "$MAX" ]
do 
echo $NUM
let "NUM += 1"

done

Until Loops

Syntax

until [ condition ]
statements
done

Example

#!/bin/bash

myvar=0

until [ $myvar -eq 10 ]
do
echo $myvar
myvar=$(( $myvar + 1 ))
done
