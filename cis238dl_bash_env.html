Preparation

Create a User

Create a new user named tux.

# useradd tux
# passwd tux

Login as user tux

# su -l tux

~/.bash_profile

Add this line to the bottom of the ~/.bash_profile file in the home directory of user tux.

echo "This line is from ~/.bashrc. The time is $(date +%R)"

~/.bashrc

Add this line to the bottom of the ~/.bashrc file in the home directory of user tux.

echo "This line is from ~/.bashrc."

~/.bash_logout

Add this line to the bottom of the ~/.bash_logout file in the home directory of user tux.

echo "Logging out as user $USER."

Login vs. Interactive Shell

Interactive Shell

As root login as user tux.

# su tux

Note that only the line from ~/.bashrc is displayed.

Type pwd and note that you are not in Tux's home directory

Type exit to logout as tux.

Login Shell

As root login as user tux using the -l option.

# su -l tux

Note that both the line from ~/.bashrc and the line from ~/.bash_profile are displayed.

Type pwd and note that you are in Tux's home directory

Type exit to logout as tux and note that the line from ~/bash_logout is displayed.
