Bash Config Files

Overview

In this assignment you will work with two Bash configuration files in your home directory. ~/.profile is the main Bash configuration script and is run when first logging into a terminal or when opening a Bash login shell using bash -l. The ~/.bashrc file is executed when opening a new Bash shell.

Tested on Ubuntu 14.01.1 LTS

Preparation

Do this assignment on the Ubuntu LiveCD or from a guest session. Click the gear icon in the upper right corner of the desktop and select Guest Session, a new Guest Session will open.

If you do this assignment in your home directory make backup copies of ~/.profile and ~/.bashrc before continuing. Clear the Bash history with history -c before starting.

Edit ~/.bashrc

  1. Click on the Dash search icon in the upper left corner of the desktop and type Gedit to bring up the Gedit text editor.
  2. Open ~/.bashrc. The file is a hidden “dot file” so right click in the Open dialog box and select Show Hidden Files. You can also use the keyboard shortcut CTRL h*.
  3. Scroll to the bottom of the file and after the last line press Enter and type echo "This line from .bashrc."
  4. Save the changes to the file.

Edit ~/.profile

  1. Open ~/.profile and scroll to the bottom of the file.
  2. After the last line press Enter and type echo "This line is from .profile"
  3. Save the changes to the file.

Edit ~/.bash_logout

  1. Open ~/.bash_logout`/
  2. After the last line press Enter and type ~echo “Say good night Gracie.”`
  3. Save the changes to the file.
  4. Close Gedit.

Start Assignment

~/.bashrc

  1. Click on Dash and type terminal. Select the Terminal icon to open GNOME Terminal.
  2. Observe that “This line from .bashrc.” shows above the prompt.
  3. Leave the Terminal open for the next step.

~/.profile

  1. At the terminal prompt type bash -l to open a new login session.
  2. Observe that both the line from .bashrc and .profile appear above the prompt.
  3. Leave the Terminal open for the next step.

~/.bash_logout

  1. Type exit to return to the previous shell.
  2. Observer that “Say good night Gracie.” displays on the screen.
  3. Type exit or CTRL d to logout of the terminal.
